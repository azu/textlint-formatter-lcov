# textlint-formatter-lcov

[textlint](https://github.com/textlint/textlint "textlint") formatter for lcov format.

## Installation

    npm install textlint-formatter-lcov

## Usage

    npm install -D textlint textlint-formatter-lcov
    textlint -f lcov
